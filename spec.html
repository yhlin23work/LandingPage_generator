文件日期：2026-06-10 ／ 專案：CMoney Landing Page Generator

1. 產品概述

Landing Page 產生器是一個單頁工具（index.html），左側為表單設定區（分頁式），右側為即時預覽區。使用者填寫產品資訊、上傳圖片後，可預覽完整 Landing Page，並一鍵下載包含 HTML、CSS 與圖片素材的完整打包檔案。亦支援匯入既有打包檔重新編輯，以及一鍵載入內建範例。

項目說明
框架純 HTML / CSS / JavaScript（無框架依賴）
外部依賴Google Fonts（Noto Sans TC / Noto Serif TC）、JSZip 3.10.1、qrcode-generator 1.4.4（皆 CDN）
佈局雙欄 Grid：左欄 480px（--panel-width，工具面板）＋ 右欄 1fr（預覽區）
面板收合可收起工具面板，僅留窄欄（--panel-collapsed-width），方便檢視桌機版完整畫面
RWD 臨界點≤ 959px：隱藏預覽區，僅顯示工具面板

3. 工具面板（左欄）

工具面板採三分頁（Tab）切換，底部 sticky action bar 提供「🗑 清空設定」、「▷ 預覽頁面」、「⬇ 打包下載」三顆按鈕。

分頁內容
內容六個可折疊區塊（Accordion），填寫頁面主要文案與圖片
顏色主色 / 輔色設定
SEO與追蹤SEO 中繼資料 ＋ 追蹤碼與事件埋點

3c. 圖片上傳與素材機制

所有圖片欄位（App Icon、Hero、功能截圖、徽章、QR Code）皆改為本機檔案上傳，不再使用 URL 輸入。上傳後以 data URL 暫存於表單，預覽時即時顯示；打包時自動轉換為 assets/images/... 相對路徑並納入 ZIP。

機制說明
一般上傳點擊上傳區選檔（PNG / JPG / WebP），可單獨清除
共用徽章App Store / Google Play 徽章預設使用內建圖（SharedImages/），可「自訂上傳」覆蓋或「還原」回預設
QR Code 自動產生autoGenerateQR()：依「App Icon ＋ 下載連結」用 qrcode-generator 產生 QR 圖；亦可自訂上傳。桌機版顯示、手機版自動隱藏

5. CTA 按鈕行為規格

Landing Page 中所有「免費下載」、「立即免費下載」按鈕均呼叫同一個 handleDownload() 函式，行為依情境判斷如下：

判斷邏輯

判斷是否在 iframe 預覽中
window.parent !== window
→
預覽中：使用 _previewMode
（由 parent 透過 postMessage 傳入）
→
非預覽（下載檔）：
UA + 視窗寬度偵測
情境判斷依據行為
桌機 預覽模式 切換按鈕為「桌機」 彈出 QR Code Modal（背景模糊，點 × / Esc / 背景關閉）
手機 預覽模式 切換按鈕為「手機」 平滑捲動至 #download 下載區塊
桌機 實際瀏覽器 視窗 ≥ 1024px 且非行動裝置 UA 彈出 QR Code Modal
手機 實際瀏覽器 視窗 < 1024px 或行動裝置 UA 平滑捲動至 #download 下載區塊
無 QR Code 圖片 表單未填 QR Code URL 兩種情境均 fallback 捲動至 #download

QR Code Modal 規格

  • 固定定位，全版背景遮罩 rgba(0,0,0,0.55) + backdrop-filter blur
  • 白底圓角卡片（border-radius: 20px），置中顯示
  • QR Code 圖片尺寸 220×220px
  • 關閉方式：右上角 × 按鈕 / 點擊背景遮罩 / 鍵盤 Esc

6. 下載打包規格

下載前確認彈窗

點「⬇ 打包下載」先開啟確認彈窗（openDownloadModal()），需完成以下項目，按鈕才會啟用：

  • 幫網頁命名：輸入頁面英文代稱（僅保留 \w-），即時預覽上架網址 https://stockhub.cmoney.tw/LandingPage/[代稱]/index.html
  • 勾選「已透過預覽畫面確認網頁內容、顏色、SEO與追蹤設定皆正確」
  • 勾選「已確認輸入的頁面英文代稱正確，且未跟其他已上架網頁重複」
※ 提醒文案：下載後先解壓縮，提供名稱為 [頁面英文代稱] 的內層資料夾請 PM Jimmy 協助上架；實際網址與 landing page 內容請於上架後再次確認。

ZIP 結構

路徑說明

[代稱]/index.html內層資料夾主檔，CSS 改為外部引用 <link rel="stylesheet" href="styles.css" />
[代稱]/styles.css完整樣式表，並於匯出時寫入最終主題色（避免配色回退預設）
[代稱]/assets/images/*上傳圖片（轉相對路徑）＋ 共用素材（SHARED_ASSETS：徽章等）

6b. 匯入與範例載入

匯入 ZIP（importZIP()）

內建範例（loadDefaultShowcase()）

範例資料夾置於 demo/ 底下，以 SHOWCASE_FOLDERS 設定表管理：

選項資料夾

起K（supertrendk）demo/LandingPage_supertrendk

選擇後 fetch 該資料夾的 index.html / styles.css，解析回填所有欄位與圖片（圖片以 setShowcaseImageAsync() 非同步抓回存入上傳暫存，確保打包正確）。
