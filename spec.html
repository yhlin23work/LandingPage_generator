1. 產品概述

Landing Page 產生器是一個單頁工具（index.html），左側為表單設定區，右側為即時預覽區。使用者填寫產品資訊後，可預覽完整 Landing Page，並一鍵下載包含 HTML、CSS 與圖片素材的完整打包檔案。

項目說明
框架純 HTML / CSS / JavaScript（無框架依賴）
外部依賴Google Fonts（Noto Sans TC / Noto Serif TC）、JSZip 3.10.1（CDN）
佈局雙欄 Grid：左欄 480px（工具面板）＋ 右欄 1fr（預覽區）
RWD 臨界點≤ 959px：隱藏預覽區，僅顯示工具面板

區塊內容說明

右欄（1fr）「預覽畫面」文字 ＋ 桌機／手機切換按鈕深灰底 #2d3240，與預覽面板對齊

3. 工具面板（左欄）

共分六個可折疊區塊（Accordion），每區塊含數個欄位。底部 sticky action bar 提供「預覽頁面」與「下載打包」按鈕。

#區塊名稱主要欄位
1基本設定產品名稱、品牌副標題、App Icon URL、頁面 <title>（SEO）、Meta description（SEO）
2Hero 主視覺Hero 主標題、Hero 介紹文字、主要下載連結（CTA URL）、Hero 圖片 URL
3Why 亮點區塊區塊標題、最多 6 個亮點（Icon / 標題 / 說明）
4特色功能區塊標題、最多 6 個功能卡片（標題 / 說明 / 截圖 URL）
5下載 CTAEyebrow 標籤、H2 標題、副標題、iOS 徽章 URL、Android 徽章 URL、QR Code URL
6頁尾資訊公司名稱、聯絡地址、版權年份

3b. SEO 設定欄位

以下欄位寫入產生頁面的 <head>，不顯示於畫面上，用途為搜尋引擎優化（SEO）。

欄位對應 HTML顯示位置預設值
頁面 <title> <title>...</title> 瀏覽器分頁標題列、Google 搜尋結果藍色標題 留空時自動使用「產品名稱｜品牌副標題」
Meta description <meta name="description" content="..."> Google 搜尋結果摘要文字（建議 100–160 字元） 留空時自動使用 Hero 介紹文字
⚠️ 這兩個欄位在頁面畫面上完全看不到，但對 Google 搜尋排名與點擊率有直接影響，建議填寫。

5. CTA 按鈕行為規格

Landing Page 中所有「免費下載」、「立即免費下載」按鈕均呼叫同一個 handleDownload() 函式，行為依情境判斷如下：

判斷邏輯

判斷是否在 iframe 預覽中
window.parent !== window
→
預覽中：使用 _previewMode
（由 parent 透過 postMessage 傳入）
→
非預覽（下載檔）：
UA + 視窗寬度偵測
情境判斷依據行為
桌機 預覽模式 切換按鈕為「桌機」 彈出 QR Code Modal（背景模糊，點 × / Esc / 背景關閉）
手機 預覽模式 切換按鈕為「手機」 平滑捲動至 #download 下載區塊
桌機 實際瀏覽器 視窗 ≥ 1024px 且非行動裝置 UA 彈出 QR Code Modal
手機 實際瀏覽器 視窗 < 1024px 或行動裝置 UA 平滑捲動至 #download 下載區塊
無 QR Code 圖片 表單未填 QR Code URL 兩種情境均 fallback 捲動至 #download

QR Code Modal 規格

  • 固定定位，全版背景遮罩 rgba(0,0,0,0.55) + backdrop-filter blur
  • 白底圓角卡片（border-radius: 20px），置中顯示
  • QR Code 圖片尺寸 220×220px
  • 關閉方式：右上角 × 按鈕 / 點擊背景遮罩 / 鍵盤 Esc

6. 下載打包規格

ZIP 結構

路徑說明
LandingPage_[產品名稱].zipZIP 根目錄（產品名稱中非字元符號自動移除）
LandingPage_[產品名稱]/index.htmlLanding Page 主檔，CSS 改為外部引用 <link rel="stylesheet" href="styles.css" />
LandingPage_[產品名稱]/styles.css完整樣式表（從 EMBEDDED_CSS 提取）
LandingPage_[產品名稱]/assets/images/*所有表單中填入的本地圖片（路徑以 assets/ 開頭）
⚠️ 外部 URL 圖片（https://...）不納入打包，保留原始 URL 引用。如本地資源不存在（fetch 失敗），該圖片靜默跳過不影響下載。

收集 assets/ 開頭路徑

→

觸發下載
